## Description

Blish is a open-sourced bridge between Guild Wars 2 API and clients. Users can add their accounts and make their profiles, characters read-only public without compromising their API Key.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
